## :sparkles:**Function**
Gamma-Ray Spectroscopy Data Visualizer (Na-22)
This Python script provides a specialized tool for plotting and analyzing radioactive decay data, specifically tailored for Sodium-22 (Na-22) histograms. 
It features a dual-axis visualization to correlate hardware voltage readings with physical energy levels (keV).

---

 ## **Features**
:white_check_mark:**Dual X-Axis Plotting:** Simultaneously displays counts against Voltage (V) and Energy (keV) for precise calibration.

:white_check_mark:**Smart CSV Parsing:** Robust data ingestion using pandas with built-in error handling for delimiters and missing values.

:white_check_mark:**Publication-Quality Graphics:** Utilizes matplotlib for clean, labeled, and color-coded spectroscopy plots.

:white_check_mark:**Isotope Specific:** Pre-configured for standard spectroscopy CSV headers (Spannung, Energie(KeV), Count).
